# Code of Conduct

## Our Pledge

We want Slipstream to be an open, welcoming, and respectful project for everyone.

Contributors, maintainers, and community members are expected to make participation in this project a harassment-free experience regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful and constructive in feedback
- assuming good intent and asking clarifying questions
- sharing context generously with new contributors
- accepting accountability and learning from mistakes
- focusing on what is best for the project and community

Examples of unacceptable behavior:

- harassment, discrimination, or personal attacks
- trolling, insulting, or derogatory comments
- deliberate intimidation or sustained disruption
- publishing private information without permission
- bad-faith engagement meant to exhaust or belittle others

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces, including:

- GitHub issues, pull requests, and discussions
- community chat or other project communication channels
- public spaces where someone is representing the project

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers at `abdullahazeemi.work@gmail.com`.

Reports will be reviewed as promptly as possible and handled with care and discretion.

## Enforcement Guidelines

Maintainers may take any action they consider appropriate, including:

- private warning
- request for behavior change
- temporary participation ban
- permanent ban from the project
